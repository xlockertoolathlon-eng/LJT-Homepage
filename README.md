# LJT-Homepage

This site was forked from the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template and is built with Jekyll and hosted on GitHub Pages.

- **Publications**: conference papers (EMNLP 2024, ICML 2024, NeurIPS 2023) and preprints
- **CV**: education, research experience, skills, publications, and awards
